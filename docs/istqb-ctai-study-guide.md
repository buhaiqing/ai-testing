```markdown
数据漂移（Data Drift）

- 阈值：PSI > 0.25 表示严重漂移

特征泄露（Feature Leakage）
- 类型：目标泄露、时间旅行、重复数据
- 检测：特征 - 目标相关性分析
- 预防：严格的数据分离
```

## 🎯 四、高频考点速记卡

### 考点 1：ML 生命周期（必考 3-5 题）

```
┌─────────────────────────────────────────┐
│          ML 系统开发生命周期              │
├─────────────────────────────────────────┤
│ 1. 数据收集 → 2. 数据准备 → 3. 模型训练  │
│      ↓                                  │
│ 5. 监控维护 ← 4. 模型评估 ← 5. 部署     │
└─────────────────────────────────────────┘

每个阶段的测试重点：
- 数据收集：数据质量、代表性、偏见检测
- 数据准备：数据清洗、特征工程、数据分离
- 模型训练：过拟合检测、超参数调优
- 模型评估：多指标评估、交叉验证
- 部署：A/B 测试、灰度发布
- 监控：数据漂移、性能衰减
```

### 考点 2：数据漂移检测（必考 2-3 题）

```
数据漂移类型：
┌──────────────┬──────────────┬──────────────┐
│   协变量漂移  │   先验漂移    │   概念漂移    │
│ (Covariate)  │  (Prior)     │ (Concept)    │
├──────────────┼──────────────┼──────────────┤
│ P(X) 变化     │ P(Y) 变化     │ P(Y|X) 变化   │
│ 输入分布变化  │ 输出分布变化  │ 输入输出关系变化│
└──────────────┴──────────────┴──────────────┘

检测方法：
- PSI（群体稳定性指标）：PSI > 0.25 = 严重漂移
- KS 检验：p-value < 0.05 = 有显著差异
- 可视化：直方图对比、t-SNE 降维
```

---

### 考点 3：神经网络覆盖度（必考 2-3 题）

```
神经元覆盖度（Neuron Coverage, NC）：

NC = 被激活的神经元数 / 总神经元数

激活判定：
- 对于 ReLU: activation > threshold (通常 0)
- 对于 Sigmoid: activation > 0.5

k-多层神经元覆盖度 (k-MNC)：
- 考虑连续 k 层的神经元激活组合
- 更严格，但计算复杂度高

注意事项：
- 高覆盖度 ≠ 高质量测试
- 覆盖度是必要非充分条件
```

### 考点 4：AI 系统质量特性（必考 3-4 题）

```
ISO 25010 传统质量特性 + AI 扩展特性：

传统特性：
- 功能性、可靠性、可用性、效率、可维护性、可移植性

AI 扩展特性（重点）：
┌──────────────┬─────────────────────────────┐
│   准确性     │ 模型预测与真实值的一致性     │
├──────────────┼─────────────────────────────┤
│   公平性     │ 不同群体的表现差异           │
├──────────────┼─────────────────────────────┤
│  可解释性    │ 决策过程的可理解程度         │
├──────────────┼─────────────────────────────┤
│   鲁棒性     │ 对抗样本和噪声的抵抗能力     │
├──────────────┼─────────────────────────────┤
│   安全性     │ 防止恶意攻击和数据泄露       │
└──────────────┴─────────────────────────────┘
```

## 📞 九、常见问题 FAQ

### Q1: 没有 ML 背景能考过吗？
**A:** 可以。CT-AI 侧重测试方法，不要求深度 ML 技术。但需要额外 2 周补 ML 基础概念。

### Q2: 必须先考 CTFL 吗？
**A:** 是的。CT-AI 是高级认证，必须持有有效的 CTFL 证书才能报考。

### Q3: 考试没通过怎么办？
**A:** 可以补考，需重新缴费。建议分析错题，针对性复习后再考。

### Q4: 证书的有效期？
**A:** ISTQB 证书终身有效，无需续费或续期。

### Q5: 中文考试准确吗？
**A:** 中文翻译质量较好，但部分术语可能不如英文准确。建议中英对照学习。
