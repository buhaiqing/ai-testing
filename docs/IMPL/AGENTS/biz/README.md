# 运营智能体需求文档

> **目录定位**：本目录包含质量智能中台**运营场景**相关的所有Agent需求文档和设计规范。

---

## 📊 运营团队角色定位

### 运营团队的核心职责

| 职责领域 | 具体工作内容 | 关键挑战 |
|---------|------------|---------|
| **数据运营** | 业务指标监控、数据分析、报表生成 | 数据质量难保证、指标口径不一致 |
| **用户运营** | 用户行为分析、留存转化优化 | 数据孤岛、分析效率低 |
| **活动运营** | 活动效果评估、ROI分析 | 数据滞后、归因困难 |
| **内容运营** | 内容质量评估、推荐效果分析 | 缺乏量化标准、反馈周期长 |
| **产品运营** | 功能使用分析、用户体验优化 | 数据采集不全、分析维度单一 |
