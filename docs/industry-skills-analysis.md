## 🏦 一、金融行业

### 1.1 行业特点

```
优势：
✅ 薪资水平高（通常高于其他行业 20-30%）
✅ 技术投入大，重视质量
✅ 职业发展前景好

挑战：
⚠️ 合规要求严格，文档工作多
⚠️ 技术栈相对保守
⚠️ 入职门槛高（学历、证书要求）
```

### 1.2 典型岗位与技能要求

#### 岗位 A：银行 AI 测试工程师

**核心技能：**

| 技能类别 | 具体要求 | 重要程度 |
|---------|---------|---------|
| **测试基础** | 银行系统测试经验、监管合规测试 | ⭐⭐⭐⭐⭐ |
| **AI 技术** | 风控模型测试、反欺诈 AI 验证 | ⭐⭐⭐⭐⭐ |
| **安全测试** | 渗透测试、数据安全、加密验证 | ⭐⭐⭐⭐⭐ |
| **编程能力** | Python/Java、SQL | ⭐⭐⭐⭐ |
| **工具栈** | Selenium、JMeter、Burp Suite | ⭐⭐⭐⭐ |

**认证要求：**
- 必选：ISTQB CT-AI
- 加分：CEH（Certified Ethical Hacker）、FRM（金融风险管理）

**典型工作内容：**
```markdown
1. 信贷风控模型验证
   - 模型准确率测试（准确率、召回率、KS 值）
   - 模型稳定性测试（PSI 指标）
   - 模型公平性测试（不同人群无歧视）

2. 反欺诈系统测试
   - 规则引擎测试
   - 实时检测性能测试（响应时间<100ms）
   - 误报率/漏报率平衡

3. 数据合规测试
   - 个人信息保护验证
   - 数据隔离测试
   - 审计日志完整性验证
```

**面试题示例：**
```
技术题：
1. 如何测试一个贷款审批 AI 模型的公平性？
2. 如何验证反欺诈系统在高并发下的表现？
3. 如何确保 AI 系统符合 GDPR/个人信息保护法要求？

场景题：
某银行的智能客服 AI 被用户投诉给出了错误的理财建议，
作为测试负责人，你应该如何分析原因并改进测试体系？
```

## 🛒 二、电商行业

### 2.1 行业特点

```
优势：
✅ AI 应用场景多（推荐、搜索、客服、物流）
✅ 技术氛围好，学习成长快
✅ 项目经验可迁移性强

挑战：
⚠️ 工作强度大（尤其大促期间）
⚠️ 技术要求全面（多端、多场景）
⚠️ 业务迭代快，测试压力大
```

### 2.2 典型岗位与技能要求

#### 岗位 A：电商 AI 测试开发工程师

**核心技能：**

| 技能类别 | 具体要求 | 重要程度 |
|---------|---------|---------|
| **推荐系统测试** | 推荐质量评估、A/B 测试设计 | ⭐⭐⭐⭐⭐ |
| **搜索测试** | 搜索相关性、排序质量评估 | ⭐⭐⭐⭐⭐ |
| **多端测试** | Web/iOS/Android/小程序全栈 | ⭐⭐⭐⭐⭐ |
| **性能测试** | 大促场景压力测试（双 11 级别） | ⭐⭐⭐⭐⭐ |
| **工具栈** | Playwright、Appium、k6、小程序测试工具 | ⭐⭐⭐⭐ |

**典型工作内容：**
```markdown
1. 推荐系统测试
   - 离线评估：Precision@K、Recall@K、NDCG
   - 在线评估：CTR、CVR、GMV 提升
   - A/B 测试设计与分析

2. 搜索质量测试
   - 查询理解准确性
   - 搜索结果相关性人工评估
   - 零结果率、坏结果率监控

3. 智能客服测试
   - 意图识别准确率
   - 多轮对话测试
   - 转人工率监控

4. 大促压测
   - 全链路压测（下单、支付、库存）
   - 限流降级策略验证
   - 应急预案演练
```

**面试题示例：**
```
技术题：
1. 如何设计一个推荐系统的 A/B 测试？
2. 双 11 大促前，你会如何规划全链路压测？
3. 如何评估搜索质量？有哪些核心指标？

场景题：
双 11 零点，推荐系统突然故障，CTR 下降 50%，
作为测试负责人，你如何快速定位问题并协助恢复？
```

## 🎮 三、互联网行业

### 3.1 行业特点

```
优势：
✅ AI 应用最前沿（AIGC、多模态）
✅ 技术成长最快
✅ 薪资天花板高

挑战：
⚠️ 技术更新快，持续学习压力大
⚠️ 工作强度高
⚠️ 35 岁危机
```

### 3.2 典型岗位与技能要求

#### 岗位 A：AI 产品测试工程师（内容平台）

**核心技能：**

| 技能类别 | 具体要求 | 重要程度 |
|---------|---------|---------|
| **AIGC 测试** | 文生图、文生视频质量评估 | ⭐⭐⭐⭐⭐ |
| **多模态测试** | 图像/语音/视频内容理解测试 | ⭐⭐⭐⭐⭐ |
| **推荐算法** | 内容推荐、个性化分发测试 | ⭐⭐⭐⭐ |
| **用户体验** | 可用性测试、用户研究配合 | ⭐⭐⭐⭐ |

**典型工作内容：**
```markdown
1. AIGC 内容质量测试
   - 图像生成质量评估（FID、人工评分）
   - 内容安全性审核（违规内容过滤）
   - 版权合规检查

2. 内容理解测试
   - 图像标签准确性
   - 视频内容分析
   - 敏感内容识别

3. 推荐效果测试
   - 用户留存率、停留时长
   - 内容分发效率
   - 信息茧房检测
```

## 🏥 四、医疗健康行业

### 4.1 行业特点

```
优势：
✅ 社会价值高
✅ 行业壁垒高（越老越吃香）
✅ 工作相对稳定

挑战：
⚠️ 入门门槛极高（医学+AI+ 测试）
⚠️ 合规要求最严格
⚠️ 责任重大（关乎人命）
```

### 4.2 核心技能要求

| 技能类别 | 具体要求 |
|---------|---------|
| **医学知识** | 基础医学、临床流程、医学术语 |

## 求职渠道
- 医疗 AI 公司（推想科技、联影智能、数坤科技）
- 互联网医疗（平安好医生、微医、丁香园）

## 📈 七、行业选择建议

### 7.1 按个人背景选择

| 你的背景 | 推荐行业 | 理由 |
|---------|---------|------|
| **金融/经济专业** | 金融科技 | 专业优势，理解业务快 |
| **医学/生物专业** | 医疗健康 | 专业壁垒高，竞争少 |
| **自动化/车辆工程** | 智能汽车 | 专业对口，行业增长快 |
| **计算机专业** | 互联网/电商 | 技术匹配度最高 |
| **文科/商科** | 企业服务/电商 | 沟通能力强，理解业务 |

### 7.2 按性格特点选择

| 性格特点 | 推荐行业 | 理由 |
|---------|---------|------|
| **追求高薪** | 金融/互联网 | 薪资天花板高 |
| **追求稳定** | 金融/企业服务 | 行业波动小 |
| **追求技术前沿** | 互联网/AI 创业 | 技术更新快 |
| **追求社会价值** | 医疗/汽车 | 直接影响生活 |
| **追求工作平衡** | 企业服务 | 加班相对少 |

### 7.3 按发展阶段选择

| 职业阶段 | 推荐策略 |
|---------|---------|
| **0-2 年（新人）** | 优先互联网/电商，快速成长 |
| **3-5 年（成长）** | 根据兴趣选择细分领域深耕 |
| **5 年 +（资深）** | 金融/汽车行业，追求稳定和高薪 |
